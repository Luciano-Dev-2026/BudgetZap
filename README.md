# 📊 Gerador de Orçamentos Fast (BudgetZap)

Solução desenvolvida para prestadores de serviços e profissionais autônomos que buscam profissionalismo e agilidade no fechamento de negócios pelo WhatsApp.

## 💡 O Problema
Muitos profissionais perdem tempo digitando preços repetitivos ou enviando listas confusas para os clientes, o que gera desistências.

## ✅ A Solução
Uma interface limpa onde o cliente (ou o próprio profissional) seleciona os serviços desejados. O sistema:
1. Soma os valores instantaneamente.
2. Gera um resumo profissional.
3. Envia o orçamento formatado diretamente para o WhatsApp do prestador com um clique.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Interface focada em usabilidade mobile)
- JavaScript (Lógica de soma e manipulação de strings para o link do WhatsApp)

## 🎯 Público-Alvo
- Técnicos de Ar-condicionado
- Manicures e Esteticistas
- Pintores e Pedreiros
- Freelancers em geral
